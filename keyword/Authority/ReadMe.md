# [Authority](https://mirror-networking.gitbook.io/docs/manual/guides/authority)

> Client authority means that the client has control of an object.  
> When a client has authority over an object it means that they can call Commands and that the object will automatically be destroyed when the client disconnects.  
Doc: https://mirror-networking.gitbook.io/docs/manual/guides/authority
